# Karnata Bandipur (Kannada + Latin Typeface)
<img width="1108" height="638" alt="image" src="https://github.com/user-attachments/assets/17d356d7-b069-44e6-87c3-b4f34bf50470" />

Karnata Bandipur is a Kannada + Latin display typeface derived from the original Bandipur project. This version focuses on improved font engineering, Kannada shaping behavior, and production readiness.

The original Bandipur typeface began as a display Latin designed by John Vargas Beltrán, and was later extended to Kannada by Dr. Priya Ravichandran. The design features a strong visual presence, with Kannada ottu characters aligned in a consistent baseline structure inspired by the Latin proportions.

## What’s New in This Version

- Improved Kannada OpenType shaping (GSUB/GPOS)
- Refined glyph structure and consistency
- Better Unicode alignment and encoding
- Cleaned and optimized font engineering
- Production-ready font files and documentation
